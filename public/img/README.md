# Image assets

Drop the real files here using these exact names. The site renders graceful
placeholders until they exist, so layout never breaks.

| File | Used for | Notes |
|------|----------|-------|
| `logo.png` | Hero logo | Transparent PNG. (A copy of the brand logo is already here.) |
| `hero-poster.jpg` | Hero video fallback | Shown instantly on slow connections / iOS / reduced-motion. ~1920×1080. |
| `service-sedan.jpg` | Services — Sedans photo | ~4:3. Placeholder box renders until added. |
| `service-suv.jpg` | Services — SUVs & Trucks photo | ~4:3. Placeholder box renders until added. |

Recommended: poster ~150–300 KB (compressed JPG), gallery images square-ish,
optimized for web.
